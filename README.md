## ✨ Features

| Feature | Details |
|---|---|
| 🌗 Dark / Light mode | Persisted via `localStorage` |
| 📱 Fully responsive | Mobile, tablet and desktop |
| 🎞️ Scroll animations | Intersection Observer reveal on every section |
| 📊 Animated skill bars | Triggered on scroll into view |
| 🔢 Counter animation | Stats count up on load |
| 📬 Contact form | Powered by [Formspree](https://formspree.io) — lands in inbox |
| 📋 Email copy | One-click clipboard copy with fallback |
| 🔗 LinkedIn link | Direct to profile |
| ⚡ No dependencies | Pure HTML/CSS/JS — zero build step |

## 🖥️ Run Locally

No build step needed — just open the file:

```bash
# Clone the repo
git clone https://github.com/birupaaaji/BIRUpaaaji.ghithub.io.git

# Open in browser
open index.html
```

Or use VS Code Live Server for hot reload.

---

## 🚢 Deploy Changes

```bash
# Stage all changes
git add .

# Commit with a message
git commit -m "your commit message here"

# Push to GitHub Pages
git push origin main
```

GitHub Pages auto-deploys within ~60 seconds of every push.

## 📝 Commit History Convention

| Prefix | Use for |
|---|---|
| `feat:` | New section or feature added |
| `fix:` | Bug fix |
| `style:` | Visual/CSS change only |
| `content:` | Text or data update |
| `refactor:` | Code restructure, no visual change |

**Example:**
```
fix: email copy bug - clipboard fallback for mobile browsers
```
